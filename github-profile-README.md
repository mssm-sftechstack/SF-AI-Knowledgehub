## How I think about SF + AI

Most people treat AI tools as autocomplete. I treat them as a junior architect who needs a proper briefing before touching production code.

Four things most AI tools get wrong on Salesforce:
- SOQL has a hard cap of 100 queries per transaction
- One trigger per object. You extend the handler, never create a second trigger.
- FLS doesn't enforce itself in Apex. You have to do it explicitly.
- Deployment order matters. Get it wrong and the whole thing rolls back.

The context templates and guardrails I'm building encode this knowledge so anyone can ship correct Salesforce code from prompt one.
